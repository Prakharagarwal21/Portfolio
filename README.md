# Portfolio Website

A modern, responsive one‑page portfolio (dark + glass UI) inspired by the sample you shared.

Built with **HTML + CSS only** (no JavaScript).

## Run locally

Option 1 (simple): open `index.html` in your browser.

Option 2 (recommended): run a small local server:

```bash
npx serve .
```

Then open the URL it prints (usually `http://localhost:3000`).

## Customize

Edit your details directly in:

- `index.html` (name, links, projects, timeline, contact)

## Add your resume

Put your PDF at `assets/resume.pdf` (the Resume buttons already point to this path).

## Deploy (Render / Netlify / GitHub Pages)

This is a static site — deploy the folder as-is.

- **Render**: create a **Static Site**, set **Publish Directory** to `.` (dot).
- **Netlify**: drag-and-drop this folder or connect a repo.
- **GitHub Pages**: push to a repo and enable Pages for the branch.
